git init.  		-> Cria repo git na pasta atual
git status  		-> Mostra status
git add  -> Adiciona arquivo para ser commitado
git commit -m  -> Faz commit do que estava para ser commitado
git log  -> Mostra todo o histórico de commit



## FLUXO DE TRABALHO

1. Criar diretório do projeto
2. Iniciar git dentro do diretório criado
3. Edita ou cria arquivos
4. git status
5. git add <arquivo>
6. git status
7. git commit -m "mensagem estranha"
8. git status
9. Volta para Passo 3
